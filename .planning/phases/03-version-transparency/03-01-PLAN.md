# Plan 03-01: Version Transparency

## Objective

Enhance version command output to show version, git commit hash, commit date, and target triple. Create `build.rs` to embed git metadata at compile time. Add update hint to both `version` and `self version` commands.

**Purpose:** Users need to know exactly what binary they're running — essential for debugging update issues and verifying deployments.

**Output:** `build.rs` file, updated `src/main.rs` with enhanced version handlers.
